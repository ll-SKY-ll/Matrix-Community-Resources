Immediate Actions

Notify server / room admins

Try reaching out to the admin of the spamming users homeserver

The most effective way to stop spam, is to try to get the account of the spamming user deactivated. Try to reach the server admin via contact info from MSC1929.

Try reaching out to the admin/moderators of the room that gets spammed

This can be very tedious, especially if the spammer just moves on to the next room after getting banned. This also wont scale and is therefore no real option for bigger instances.


Cleanup the spam

Delete remote media

Synapse

Use the "Purge remote Media" admin API to delete all media from remote servers.

üí° This is not as destructive as it sounds, media like avatars or other images in chats will be re-downloaded from the original server as soon as a client requests it again. Youre only deleting your local cache

Precautions

Media handling

Synapse

Set up remote media retention

Use the "media_retention" config options and configure the remote media lifetime.

Exclude remote media from your backups

In the directory that is configure in "media_store_path" there is a subdirectory called "remote_content". You should exclude this from your backups to prevent possibly illegal material from making its way into your backups (if you restore from backup, you will have to use the "Purge remote Media" admin API" to tell synapse that the media is gone, otherwise your remote media will be broken).

Check whats happening on your server

Synapse

Check the rooms

Most of the time the illegal content is unwanted, but some of your users might intentionally participate in rooms with the sole purpose of distributing illegal content. In that case, the best way to prevent illegal content on your server is to deactivate these users and blocking the questionable rooms to prevent future users from even joining them in the first place.

List rooms via the "list_rooms" API, block rooms via the "block-room" API and deactivate accounts with the "Deactivate account" admin API.
Alternatively use Synapse Admin for all of these tasks.
